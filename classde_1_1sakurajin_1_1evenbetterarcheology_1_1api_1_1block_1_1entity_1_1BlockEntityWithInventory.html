Detailed Description

A simple SidedInventory implementation with only default methods + an item list getter.

Reading and writing to tags

Use Inventories#writeNbt(NbtCompound, DefaultedList) and Inventories#readNbt(NbtCompound, DefaultedList) on the item list.
